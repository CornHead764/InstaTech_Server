InstaTech Server Change Log

Version 1.8.4

  • Clients updated.
  • Changed scripting command behavior.
  • Modified error log path.
  • Added separate override folder for customizing the remote control.

Version 1.8.3

  • Modified save procedures to fix computer group disappearing.
  • Simplified authentication token model. Only last 10 are saved now, instead of last 30 days.

Version 1.8.2

  • Fixed custom image persistence through updates.

Version 1.8.1

  • Added CSS and JS overrides.
  • Added loading wheel when performing quick connect from Computer Hub.

Version 1.8.0

  • Added a notification system to the tech portal.
  • Removed SSL requirement. A warning will now display on the clients and the tech portal notification menu.
  • Added server version checking. If an update is available, it will appear in the notification area.
  • Updated packages can now be downloaded and installed from the above notification.

Version 1.7.2

  • Completed Reference guide.
  • Fixed Account Center bug when saving.
  • Removed remaining references to licensing.

Version 1.7.1

  • Fixed bug with quick connect from Computer Hub.

Version 1.7.0

  • Added support for multiple concurrent authentication tokens. This means you can cache your credentials (i.e. check "Remember Me") in multiple browsers. They expire after a month of not being used.

Version 1.6.0

  • Added session recording.

Version 1.5.0

  • Open-sourced client and server.
  • Removed license and trial checks.

Version 1.4.5

  • Removed WebRTC from Electron clients.
  • .NET clients now work on Windows 7.

Version 1.4.3

  • Changed data transfer process to improve performance on slower connections.
  • Increased maximum virtual screen size.
  • Minor UI changes.

Version 1.4.2

  • Added persistent sessions for interactive console.

Version 1.4.1

  • Added console command history (up and down arrows).
  • Fixed a bug with escaping quotes in interactive PowerShell console.

Version 1.4.0

  • Added interactive scripting console.
  • Added Last Logged-On User to search matches.
  • Fixed bug with process not exiting when deploying a file.

Version 1.3.1

  • Fixed computer hub scrolling after new search.
  • Fixed a bug that would allow multiple connections from the same account to remain open.
  • Fixed horizontal scrolling on computer hub.

Version 1.3.0

  • Initial release.
