InstaTech Service Change Log

Version 1.5.3

  • Updated packages.
  • Bug fixes related to move to Azure.

Version 1.5.2

  • Modified heartbeat and error recovery.

Version 1.5.1

  • Added ability to customize icon, background image, company name, and websocket port.

Version 1.5.0

  • Removed SSL requirement. A warning will now display if the connection isn't secure.

Version 1.4.4

  • Replaced JSON.NET with custom JSON class.
  • Fixed bug with key combinations using Shift, Control, and Alt modifiers.

Version 1.4.3

  • Fixed bug with unattended service when no instance of explorer.exe is running.

Version 1.4.2

  • Added DPI awareness.
  • Removed auto-updating for one-time service upgrades.

Version 1.4.1

  • Fixed a bug with multi-monitor setups.

Version 1.4.0

  • Now works on Windows 7, thanks to a new websocket library.

Version 1.3.6

  • Changed data transfer process to improve performance on slower connections.
  • Increased maximum virtual screen size.

Version 1.3.5

  • Fixed auto-update bug.
  • Improved cursor position accuracy on computers without monitors.

Version 1.3.4

  • Improved reliability of uninstall process.
  • Removed redundant -update switch. -Install does both.

Version 1.3.3

  • Fixed a bug with the auto-update feature of the service client.

Version 1.3.2

  • Added persistent sessions to the interactive console.

Version 1.3.1

  • Fixed a bug with escaping quotes in interactive PowerShell console.

Version 1.3.0

  • Added interactive scripting.
  • Fixed a bug with file deployment processes not exiting.
  • Fixed a bug when sending clipboard via Windows service.
  • Increased file size limit of remote control file transfer.
  • Fixed a bug that could cause the session to disconnect for a non-elevated client when the lock screen or UAC desktop is opened.

Version 1.2.0

  • Merged the InstaTech Service into the InstaTech Client.
  • Added "Upgrade to Service" in the options menu. This will perform a one-time upgrade, and the service is removed afterward.
  • Added "Unattended Access" in the options menu to install/uninstall the service.
  • Added -install, -uninstall, and -update command line switches for controlling the service.
  • Added UAC handling to prevent cursor from getting stuck on UAC prompts (which occurs when not running as a service).
  • Added notification banner that displays when remoted in via Unattended Mode.
  • Miscellaneous UI fixes and tweaks.

Version 1.1.0

  • Added "follow cursor" to automatically scroll browser window.
  • Added Ctrl + Alt + Del.
  • Fixed wheel input.
  • Other refactorings.

Version 1.0.2

  • Official release.
  • Added more keybinds.
  • Bug fixes.

Version 0.9.0

  • Initial beta release.
